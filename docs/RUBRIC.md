# Evaluation Rubric

## Functionality (0–5)
How well the app runs and whether the core slot machine logic works.

## User Experience (0–5)
How easy and clear the app is to use, including layout and feedback.

## Completeness (0–5)
How complete the app feels, including presence of key features like tokens, spinning, and win/loss behavior.

## Code Quality (0–5)
How readable, organized, and structured the code is.

## Creativity / Theme (0–5)
How well the app incorporates the AI theme and overall creativity.
